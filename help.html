Help for CVI Tool

Basically, the table on the left is where you put the people profiles. On the right, you'll see the profile diagram (once one is drawn).

Adding a Profile

You can add a profile directly. In the 5 boxes at the bottom of the table, enter the relative information based on the column. Then click the button on the left end of the same row. That will add your new person to the table.

Repeat as desired.

Removing a Profile

If you've accidentally added a profile incorrectly, you can click on the on the right-end of that row to remove it.

Exporting Profile Data

To save you some typing, if you click the button, it will prompt you to save a file which is the table data in a convenient format (... for the tool, not you)

Importing Profile Data

You can re-load the exported data by using the button and selecting the saved file. The file format is a JSON array of arrays.

Display a Profile

Now, to display a single person's profile, click on the little grey check-box () to the left of the name in the table. You should see the "diagram" of the CVI profile appear on the right (or possibly below if your browser window is small)

Group Profile

If you click on additional people's checkboxes, the tool will "overlay" each CVI profile on the same drawing. You will see an internal "profile" with a grey box inset on the diagram. This is the "group average" profile -- so you can see if your group is decently balanced. (Ideally it should be square, centered on the middle of the box)

Save The Diagram

Lastly, if you want to save the drawn diagram, select the type of image you'd like to create in the drop-down menu on the bottom-right of the table The default format SVG (scalable vector graphics) and can also export PNG and JPEG. Once you select the format, click on and you'll download the file.
